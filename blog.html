My first blog post for programming!

June 22, 2018

Hi! I'm so happy that you stopped by. In the age of WordPress and Blogger, That's still a thing, right?, I decided to write my blog the old fashioned way. That's right, I'm using plain old HTML. I think it's a great way to work on my HTML chops.

My hope with this blog is to share the fun new programming knowledge and skills I have been gaining with the We Can Code IT Flexcode™ bootcamp that I am fortunate to have been sent to by my current employer, Bartha. The end goal is for me to be able to write custom applications for the company. While the training is for Junior Developer position, I'll be needing to step up to Senior Developer once the course is complete.

So, at this point we are few weeks into the course and after having spent a few weeks learning Java, we're currently working on learning front-end languages such as HTML, CSS, and Flexbox. I'm looking forward to diving into some JavaScript and everthing else they have planned for us!
